# go-homeworks

Репозиторий с примерными решениями домашних заданий по курсу Go.

- Решения выкладываются **только после истечения дедлайна**.
- Код можно использовать как подсказку и источник идей.
- Не копируйте решения без понимания — это мешает учиться.

## Структура

- `month01`, `month02`, ... — месяцы (разделы курса).
- `lesson01`, `lesson02`, ... - уроки раздела
- Внутри каждого месяца — папки с домашними заданиями:
  - `intro-basics`, `goroutines-basics`, `http-basics`, ...

В каждой папке ДЗ есть:
- `README.md` — краткое описание задания и как запускать решения.
- Подпапки с решениями (`cls`, `flag`, `solution1`, `solution2`, ...), внутри — `main.go` или несколько файлов.

## Как запускать

Переходите в нужную папку с решением и запускайте:
```bash
cd month03/lesson01-intro/cls
go run ./...
